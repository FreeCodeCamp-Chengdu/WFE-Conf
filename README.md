## 技术栈

-   逻辑语言: [TypeScript v3][4]
-   组件引擎: [WebCell v2][5]
-   组件库: [BootCell][6]
-   状态管理: [MobX v5][7]
-   打包工具: [Parcel v1][8]

## 本地开发

### 安装工程

```shell
git clone https://github.com/FreeCodeCamp-Chengdu/Web-Conf.git

npm install

npm start
```

### 导入素材

```shell
node toolkit/import-assets path/to/media/folder
```
