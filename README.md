## 技术栈

-   逻辑语言: [TypeScript v3][3]
-   组件引擎: [WebCell v2][4]
-   组件库: [BootCell][5]
-   状态管理: [MobX v5][6]
-   打包工具: [Parcel v1][7]

## 本地开发

```shell
git clone https://github.com/FreeCodeCamp-Chengdu/Web-Conf.git

npm install

npm start
```
